# Vouch ✅
Don't share it until you Vouch for it.
Vouch is an AI-powered Chrome extension that fact-checks any claim on the internet in one click. Highlight text, right-click, and instantly get a verdict backed by real sources, a trust score, and a plain-English explanation — right where you need it.

# Features

Chrome Extension

- Paste & Analyze: Paste any claim, headline, or text and get an instant AI verdict
- Scan This Page: Automatically detects the top 3 fact-checkable claims on any webpage
- Website Credibility: Rates the credibility of the site you're currently visiting
- Right-click Fact Check: Highlight any text, right-click, and select "Vouch this claim"
- Bias Meter: Visual left/right political spectrum bar for every analysis
- Echo Chamber Detector: Warns when all sources share the same ideological bias
- Virality Risk Score: Predicts how likely a false claim is to spread
- Counter-Narrative Generator: Generates a shareable correction for false/misleading claims
- Claim History: Logs every claim you've checked with verdict and timestamp
- Vouch Score: Your personal credibility engagement score

# How It Works

1. Highlight any claim on the web, or paste text into the extension
2. Vouch analyzes it — checking against reliable sources and evaluating bias
3. Get the truth — verdict, trust score, sources, bias analysis, and plain-English explanation
